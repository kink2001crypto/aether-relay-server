# Changelog

## [2.3.0] - 2024-02-02

### Added
- Multi-platform compatibility: VS Code, Cursor, VSCodium, Code-OSS, Windsurf, Antigravity
- Open VSX Registry support for non-Microsoft marketplace distribution
- Untrusted workspace support
- Improved IDE detection for better cross-platform experience

### Changed
- Lowered minimum VS Code engine version to 1.70.0 for broader fork compatibility
- Updated TypeScript definitions to match broader compatibility
- Enhanced .vscodeignore for cleaner VSIX packages

### Fixed
- TreeDataProvider event firing for older VS Code versions

## [2.2.0] - 2024-01-11

### Added
- Cloud sync with Fly.io server
- Mobile project management panel
- File content caching for offline mobile access

## [2.1.0] - 2024-01-10

### Added
- Live terminal sync with mobile
- Git operations (status, commit, push)
- Project file sync

## [2.0.0] - 2024-01-07

### Added
- Initial release with AI chat panel
- Multi-model support (Gemini, Claude, OpenAI, Ollama, DeepSeek, Grok)
- File editing from mobile
